# Phase 9 Codex Task

Implement refinement and production polish for the bilingual personal website.

## Goal
Unify the visual language of the whole site and improve the quality of layout, motion, readability, and mobile behavior.

## Priorities
1. Improve global spacing and typography consistency.
2. Refine flagship story pages for Elementization and ALOA.
3. Improve mobile fallback behavior for story pages.
4. Strengthen supporting pages so they feel consistent with the rest of the site.
5. Add a small set of reusable premium UI blocks for technical content and contact presentation.

## Tasks
1. Refine spacing, max-widths, and vertical rhythm across the site.
2. Improve typography hierarchy and readability for both English and Chinese.
3. Improve header/footer polish and active-state behavior.
4. Refine story progress indicator styling and behavior.
5. Refine GSAP/scroll timing where needed on flagship pages.
6. Add reusable UI components such as:
   - metric strip
   - callout card
   - link card
   - contact method card
   - section heading helper
7. Improve About, Research, Accomplishments, and Contact pages using these reusable components.
8. Keep everything modular and mobile-friendly.

## Constraints
- do not change route structure
- do not add new major libraries
- do not make the site visually noisy
- do not overcomplicate supporting pages
- do not remove bilingual support

## Expected result
- stronger visual consistency
- cleaner typography
- better mobile behavior
- more premium supporting pages
- more polished flagship page experience
