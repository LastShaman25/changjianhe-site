# Phase 7 Codex Task

Implement motion foundations and interactive homepage behaviors for the bilingual personal website.

## Goal
Add tasteful, premium motion to the homepage without introducing heavy scroll-scene complexity yet.

## Tasks
1. Install and use Framer Motion for homepage interactions.
2. Create reusable motion helpers/components under `src/components/motion/`.
3. Add reveal animations for homepage sections.
4. Add staggered entrances where appropriate.
5. Add subtle interactive motion to featured project cards.
6. Add tasteful button motion for primary CTAs.
7. Keep motion premium, restrained, and mobile-friendly.
8. Preserve locale-aware routing and existing component structure.

## Suggested implementation areas
- Hero section
- Identity strip
- Featured projects
- Philosophy section
- Accomplishments preview
- Contact CTA

## Constraints
- do not introduce GSAP timeline scenes yet
- do not redesign the layout system
- do not change route structure
- do not add noisy or flashy animations
- do not break mobile usability
- respect reduced-motion preferences where practical

## Expected result
- homepage feels more alive and premium
- sections animate in cleanly
- project cards feel interactive
- code remains modular and ready for later cinematic phases
