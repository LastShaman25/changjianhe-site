# Phase 10 Codex Task

Implement deploy readiness and launch setup for the bilingual personal website.

## Goal
Prepare the site for production launch on Vercel with a custom domain, metadata, social sharing, crawlability, and a working contact flow.

## Tasks
1. Add production-ready metadata to the app.
2. Add Open Graph and Twitter sharing support.
3. Add `src/app/sitemap.ts`.
4. Add `src/app/robots.ts`.
5. Add favicon/app icon support.
6. Implement a working contact submission flow.
7. Prepare environment variable usage for production.
8. Keep the implementation compatible with Vercel deployment.
9. Preserve bilingual support and current route structure.
10. Add or improve final page metadata where useful.

## Constraints
- do not change route structure
- do not add unnecessary libraries
- keep the contact implementation clean and minimal
- preserve the current visual direction
- keep deployment simple and maintainable

## Expected result
- launch-ready metadata
- working sitemap and robots
- social sharing support
- contact flow ready for production
- Vercel-ready deployment state
